# florida-restaurants-starter
Construir un “mapa vivo” 2000–2025 del ecosistema de restaurantes en Florida, unificando datos de Yelp, Google y Overpass/OSM en un repositorio limpio y deduplicado (SSOT) para: análisis y dashboards (densidad, tendencias, cocinas, precios, reputación) y modelos predictivos (riesgo de cierre, proyección de reseñas).
